スクールバス

　スクールバスの運行については，平成３０年度は，下記の５方面７コースでの運行を行っています。

運行コース

古河市内方面

  • 古河市内～学校

    • 古河市大山 → 古河四小前 → 坂間 → 鴻巣 → 長谷町 → 本町 → 松並 → 三杉町 → 東本町 → 古河駅東口 → 旭町 → 学校

坂東市・境町方面

  • 坂東市・境町～学校（坂東・境①コース）

    • 坂東市小山 → 長谷 → 岩井 → 鵠戸 → 長須 → みむら → 境町伏木 → 大歩 → 学校

  • 坂東市・境町～学校（坂東・境②コース）

    • 坂東市矢作 → 矢作 → 中里 → 辺田 → 大山 → 岩井 → 弓田 → 沓掛 → 学校

  • 坂東市・境町～学校（坂東・境③コース）

    • 坂東市山 → 菅谷 → 境町山崎 → 松岡町 → 塚崎 → 学校

下妻・八千代方面

  • 下妻市・八千代町～学校

    • 下妻市宗道 → 田下 → 下妻駅東口 → 長塚 → 八千代町今里 → 沼森 → 若 → 村貫 → 平塚 → 坂東市逆井 → 境町内門 → 蛇池 → 猿山 → 古河市谷貝 → 学校

結城市方面

  • 結城市～学校（結城①コース）

    • 結城駅南口 → 新福寺 → 結城 → 結城作 → 田間 → 平間 → 今宿 → 東茂呂 → 学校

  • 結城市～学校（結城②コース）

    • 結城市北南茂呂 → 下七五三場 → 古河市諸川 → 下片田 → 上大野 → 小堤 → 学校

筑西市方面

  • 筑西市～学校

    • 下館駅南口 → 玉戸 → 布川房山 → 舟生 →下妻市江 → 黒駒 →八千代町高崎 → 久下田 → 佐野 → 古河市東間中橋 → 尾崎 → 恩名 → 東山田 → 山田 → 葛生 → 学校

　スクールバスは，保護者団体であるスクールバス運行運営会とバス会社との契約によって運行されます。

　運賃は国土交通省の新運賃制度のもと，学校からの距離を考慮して設定されます。
　１ヶ月あたりの運賃は，古河市内方面で23,000円程度，その他方面で25,000円～33,000円程度となります。

　スクールバスの運賃は，利用者の状況やその他の運行条件をもとに，毎年見直しを行うことになります。
